# Mobil

## Technológiák

- **React Native** (v0.81.5)
- **Expo** (~54.0.32)
- **React** (19.1.0)
- **React Navigation** (navigációs könyvtárak)

## Installáció és futtatás

```bash
# Függőségek telepítése
npm install

# Csomagook telepítése (expo)
npx expo install {name}

# Projekt inditasa
npm run start

```
