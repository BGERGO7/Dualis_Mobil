## Áttekintés
Ez a ként program (Frontend es Backend) **jelenleg két különböző feladatot lát el**. Az Frontend React Native segítségével létrehoz egy mobil alkalmazást, ahol 20 db mockold adatot rendez el egy listában. A Backend alkalmazás Express.js segítségével cikkeket kezel. 

## Projekt célja
A későbbiekben ezt **a két alkalmazást össze fogjuk kötni**, hogy a backend által lekérdezett cikkek megjelenjenek a React Native-es alkalmazásban. Egy **infinite scroll appot fogunk létrehozni**, hasonló mint a Twitter (X). Ez a projekt tökéletes arra, hogy gyakorlatban lássuk, hogy **hogy működik a Frontend és a Backend együtt egy real-life alkalmazásban**.
